# Glass Fracture AI Solver 🔬

An AI-powered physics engine that calculates optimal drop height and angle to achieve specific glass fracture patterns based on target distance.

## 🚀 Features
- **Inverse Neural Network:** Solves complex physics variables.
- **Live UI:** Built with Streamlit for instant testing.
- **Visualization:** Shows the impact zone for the predicted variables.

## 🛠️ Tech Stack
- Python
- PyTorch (AI Model)
- Streamlit (Web App)
- Scikit-learn (Data Scaling)
